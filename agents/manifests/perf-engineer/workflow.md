# Performance Engineer — Workflow

1. Definir baseline de performance actual
2. Identificar bottlenecks (CPU, memory, I/O, network)
3. Load test con k6 o artillery
4. Profiling de código (py-spy, cProfile)
5. Database query optimization (EXPLAIN ANALYZE)
6. Cache strategy (Redis, CDN, HTTP caching)
7. Medir mejora, iterar
