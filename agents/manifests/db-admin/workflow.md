# DB Admin — Workflow

1. Entender el modelo de datos y queries
2. Diseñar esquema (normalización hasta 3NF)
3. Migraciones versionadas (nunca tocar la DB directo)
4. Índices estratégicos (no sobre-indexar)
5. Connection pooling configurado
6. Backup automático (WAL + full semanal)
7. Monitoreo de slow queries
