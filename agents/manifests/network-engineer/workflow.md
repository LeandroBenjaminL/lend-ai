# Network Engineer — Workflow

1. Diseñar arquitectura de red
2. Configurar DNS (registros A, CNAME, MX, TXT)
3. TLS/SSL con Let's Encrypt (auto-renovación)
4. Reverse proxy con Nginx/Caddy
5. Firewall rules (whitelist, no blacklist)
6. Load balancing si hay múltiples instancias
7. Monitoreo de tráfico y latencia
