# Git/GitHub — Patterns

## Patrón de commit
tipo(scope): mensaje en imperativo (< 50 chars)

Cuerpo opcional con detalles (< 72 chars por línea)

Closes #123

## Patrón de branch
tipo/issue-numero-descripcion
feat/42-login-auth
fix/17-crash-on-null

## Patrón de PR
- Descripción: qué, por qué, cómo, screenshots
- Labels: feat, fix, chore, breaking
- Review: asignar revisor
- CI: debe pasar antes de mergear
