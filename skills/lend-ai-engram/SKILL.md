# Skill: lend-ai-engram

Cargá esta skill cuando necesites guardar o consultar información en Engram.

## Qué guardar

- Decisiones de arquitectura
- Bugs y fixes
- Patrones y convenciones
- Configuraciones y setup
- Descubrimientos técnicos (gotchas, edge cases)
- Resúmenes de sesión

## Formato

Toda entrada usa: **What**, **Why**, **Where**, **Learned**

## Cuándo guardar

- Después de cada decisión significativa
- Al encontrar y corregir un bug
- Al finalizar una sesión
- Cuando descubrís algo que puede ser útil después

## Cuándo consultar

- Al empezar una sesión
- Antes de tomar decisiones de arquitectura
- Cuando un bug parece familiar
